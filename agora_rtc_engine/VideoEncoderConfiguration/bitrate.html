bitrate property

int bitrate
read / write

The bitrate of the video.

Sets the video bitrate (Kbps). If you set a bitrate beyond the proper range, the SDK automatically adjusts it to a value within the range. You can also choose from the following options:

  • Standard: (recommended) In this mode, the bitrates differ between the Live-broadcast and Communication profiles:
  • Communication profile: the video bitrate is the same as the base bitrate.
  • Live-broadcast profile: the video bitrate is twice the base bitrate.
  • Compatible: In this mode, the bitrate stays the same regardless of the profile. In the Live-broadcast profile, if you choose this mode, the video frame rate may be lower than the set value. Agora uses different video codecs for different profiles to optimize the user experience. For example, the Communication profile prioritizes the smoothness while the Live-broadcast profile prioritizes the video quality (a higher bitrate). Therefore, Agora recommends setting this parameter as AgoraVideoBitrateStandard.

Implementation

int bitrate = AgoraVideoBitrateStandard
